Tester for the libft project of 42 school (with personalized leaks checking on mac, using valgrind on linux) ~\newline
 Clone this tester in your libft repository, or somewhere else and customize the path to your libft project by changing the LIBFT\+\_\+\+PATH variable inside the Makefile. ~\newline

\hypertarget{md_libftTester-master_2README_autotoc_md8}{}\doxysection{\texorpdfstring{Commands}{Commands}}\label{md_libftTester-master_2README_autotoc_md8}
make m = launch mandatory tests ~\newline
 make b = launch bonus tests ~\newline
 make a = launch mandatory tests + bonus tests ~\newline
 make \mbox{[}function name\mbox{]} = launch associated test ex\+: {\ttfamily make calloc} ~\newline


make dockerm = launch mandatory tests in linux container ~\newline
 make dockerb = launch bonus tests in linux container ~\newline
 make dockera = launch mandatory tests + bonus tests in linux container ~\newline
 make docker \mbox{[}function name\mbox{]} = launch associated test in linux container ex\+: {\ttfamily make dockercalloc} ~\newline
 Thanks to gurival-\/ for the docker idea (\href{https://github.com/grouville/valgrind_42}{\texttt{ https\+://github.\+com/grouville/valgrind\+\_\+42}}) ~\newline


make vs \mbox{[}function name\mbox{]} = open the corresponding tests in vscode ex\+: {\ttfamily make vscalloc} ~\newline
